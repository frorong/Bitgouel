# Bitgouel

### Description

Bitgouel은 빛고을 배드민턴 클럽의 홈페이지입니다.

### GET start

```
- git clone
$ git clone https://github.com/frorong/Bitgouel.git

- setting project
$ pnpm install

- start project
$ pnpm dev

- build
$ pnpm build
```
